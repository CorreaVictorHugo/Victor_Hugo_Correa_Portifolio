# 🚀 Portfólio Profissional — Victor Hugo Nunes Correa

Este repositório contém o código-fonte do meu site de portfólio, desenvolvido com o objetivo de apresentar minha trajetória, projetos e evolução técnica na área de Dados e Engenharia de Dados.

🔗 Acesse o site:  
https://victor-hugo-correa.vercel.app/

---

## 📌 Sobre o Projeto

O portfólio foi construído com foco em clareza, organização e narrativa técnica, permitindo apresentar:

- Minha transição de carreira para a área de tecnologia
- Experiência profissional com automação de processos
- Projetos práticos em Engenharia de Dados e Business Intelligence
- Stack tecnológica utilizada no dia a dia

O site foi desenvolvido com uma identidade visual minimalista em modo escuro, priorizando legibilidade, performance e experiência do usuário.

---

## 🧠 Objetivo

Criar uma vitrine profissional que vá além de um currículo tradicional, demonstrando na prática:

- Capacidade de estruturar informações
- Organização de projetos técnicos
- Aplicação de conceitos de dados em cenários reais
- Comunicação clara de soluções técnicas

---

## 🛠️ Tecnologias Utilizadas

### Front-end
- HTML5
- CSS3 (com variáveis e responsividade)

### Conceitos Aplicados
- Design responsivo (Mobile First)
- Organização de layout em grid
- Estruturação de componentes reutilizáveis
- Animações com Intersection Observer
